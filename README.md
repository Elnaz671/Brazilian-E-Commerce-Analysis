# Brazilian-E-Commerce-Analysis
E-commerce data analysis project using Python and Power BI

## Project Overview
This project analyzes Brazilian e-commerce data using Python and Power BI.

The goal of this project is to explore:
- Customer behavior
- Payment methods
- Top-selling categories
- Revenue trends
- Seller performance

---

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Power BI

---

## Analysis Performed
- Exploratory Data Analysis (EDA)
- Revenue Analysis
- Payment Analysis
- Monthly Sales Trend
- Top Sellers Analysis
- Dashboard Visualization

---

## Dashboard Preview
Power BI dashboard includes:
- Total Revenue
- Total Orders
- Payment Distribution
- Top Categories
- Monthly Sales Trend
- Top Sellers
